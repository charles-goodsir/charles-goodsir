## Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation
```bash
# Clone the repository
git clone https://github.com/charles-goodsir/my-portfolio.git

# Navigate to the project directory
cd my-portfolio

# Install dependencies
npm install

# Start the development server
npm run dev
```
